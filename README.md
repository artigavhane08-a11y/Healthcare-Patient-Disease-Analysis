# Healthcare-Patient-Disease-Analysis
🏥Healthcare Patient & Disease Analysis

📊 Project Overview

This project analyzes healthcare data to understand patient distribution, disease trends, billing insights, and hospital performance.

🔑 Key Insights

- Total Patients: 55.5K
- Average Age: 51.54
- Major Diseases: Diabetes, Arthritis, Hypertension
- Equal Gender Distribution (~50% Male & Female)
- Billing distributed across multiple insurance providers

📈 Dashboard Features

- Patient Age Group Distribution
- Disease Count Analysis
- Blood Type Distribution
- Admission Trends (Quarterly)
- Billing by Insurance

🛠 Tools Used

- Power BI
- Excel / CSV
- Data Visualization

📌 Conclusion

This dashboard helps in understanding healthcare patterns and supports better decision-making.
